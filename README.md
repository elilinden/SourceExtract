# SourceExtract
**The ultimate web media extractor and browser for iOS.**

Welcome to the official support and marketing page for **SourceExtract**, available soon on the App Store.

## About the App
SourceExtract is a beautifully designed iOS browser that goes beyond standard web surfing. Built natively with SwiftUI, it features a specialized extraction engine that allows you to effortlessly discover, collect, and save images and videos from any web page.

Whether you are collecting references for a design project, archiving assets, or just browsing, SourceExtract's powerful deep-scan technology ensures you never miss a hidden file. 

## Key Features
* **Deep Media Collection:** Powered by a custom JavaScript extraction engine, the app scans `<img>` tags, `<video>` tags, CSS backgrounds, shadow DOMs, and performance resources to uncover media that standard browsers miss.
* **Smart Deduplication:** Automatically filters out duplicate URLs, presenting you with a clean, organized gallery of unique assets.
* **Modern Browsing Experience:** Navigate the web through a seamless, glass-morphic interface complete with a multi-tab manager, bookmarks, and local history.
* **Batch Downloading:** Queue and manage downloads directly to your device's photo library or files with a dedicated download manager.
