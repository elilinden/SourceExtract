Discovery Engine

Four ways to find media

Most download tools only see what's visible on the surface. SourceExtract goes deeper with four independent discovery methods.

🖼️

IMG & Picture Tags

Standard images, srcset variants, SVG files, and picture elements extracted directly from page HTML.

🎬

Video Tags

Embedded video elements and all associated source files detected across the page DOM.

🎨

CSS Backgrounds

Images applied through stylesheets, discovered by scanning up to 2,500 page elements.

🌐

Network Resources

Media loaded during the page lifecycle, captured via the Performance API.

The scanner also penetrates Shadow DOM, traverses same-origin iframes, and resolves blob and data URLs — so nothing slips through.

Results

Preview, filter & sort

Browse your results with tools that make it easy to find exactly what you're looking for.

Browse & organize

Once the scan completes, browse results in a clean grid or list view. Filter by images, videos, or all media. Sort by page order, largest first, or smallest first.

Each item shows its dimensions, file size, and a badge indicating how it was discovered.

Full-size preview

Tap any result for a full-size preview with pinch-to-zoom, swipe navigation between items, and one-tap options to share or copy the URL.

Downloads

Download everything at once

A built-in download manager designed for speed and reliability.

Batch downloads

Select the items you want — or select all — and download in a single tap. The built-in download manager runs concurrent downloads, shows real-time progress, and validates every file to ensure you're saving actual media and not error pages.

Any failed downloads can be retried individually.

Save your way

When your downloads finish, save directly to your Photos library or share through the system share sheet.

Your files, your choice.

Organization

Bookmarks & collection history

Save pages and revisit past scans without re-browsing.

Bookmark pages you return to often for instant access. SourceExtract also keeps a history of every scan, including a snapshot of what was found — so you can revisit results from any page you've previously scanned without having to browse back to it.
